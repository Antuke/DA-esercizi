# Esercizi di Data Analysis

Questa cartella contiene una raccolta di esercizi e soluzioni per il corso di Data Analysis, basati su tracce di esami passati.

## Contenuto

*   **Script R (`.R`)**: Soluzioni implementate in R per analisi statistica, regressione e selezione del modello.
*   **Script Python (`.py`)**: Soluzioni implementate in Python per classificazione e regressione.

## Argomenti Trattati

*   **Regressione Lineare e Polinomiale**: Stima dei parametri, intervalli di confidenza e previsione.
*   **Model Selection**: Stepwise selection (Forward/Backward/Exhaustive), criteri BIC e Adjusted R-squared.
*   **Regolarizzazione**: Ridge e Lasso Regression.
*   **Classificazione**: Logistic Regression, KNN, MAP classifier.
*   **Cross-Validation**: Implementazione manuale e tramite librerie per la stima dell'errore.

## Utilizzo

Gli script sono pronti per l'esecuzione. Assicurarsi di avere installate le librerie necessarie:
*   **R**: `corrplot`, `leaps`, `pracma`, `car`, `glmnet`.
*   **Python**: `numpy`, `scipy`, `matplotlib`.
